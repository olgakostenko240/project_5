# Проект project_5

## Описание проекта:

Проект project_5 - создан как учебный, для работы HTML-кодом с помощью Bootstrap ,для создания проекта в Django.

##  Установка

1. Клонируйте репозиторий:
```
git@github.com:olgakostenko240/project_5.git
```
2. Установите зависимости:
```
poetry install
```
3. Установите пакеты для работы с Django
```
poetry add django
```
4. Установите дополнение для работы с Django
```
poetry add djangorestframework
```

## Использование:

1. Создано новое приложение materials.
2. В приложении materials созданы модели well и lesson.
3. Создано новое приложение users.
4. В приложении users создана модель user.
5. Для модели well реализован  CURD используя Viewsets
6. Для модели lesson реализован  CURD используя Generic-классы.
